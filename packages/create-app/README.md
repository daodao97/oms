## @vue-oms/create-app

`vue-oms` 的项目模板.

```shell
npm init @vue-oms/app
# or
yarn create @vue-oms/app

# npm 6.x
npm init @vue-oms/app my-oms-app --template ts 

# npm 7+, extra double-dash is needed:
npm init @vue-oms/app my-oms-app -- --template ts

# yarn
yarn create @vue-oms/app my-oms-app --template js
```

template 的可选项有: `js`, `ts`
