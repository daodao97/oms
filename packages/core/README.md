## `vue-oms`

`vue-oms: Vue Opearator Manager System` 是基于`vue`的中后台管理系统脚手架, 通过 核心包 `@vue-oms/core`
和 扩展包 `@vue-oms/plugin-**` 的方式, 任意组合脚手架和扩展包功能.

## `@vue-oms/core`

此包为核心包, 提供后台管理系统的基础功能, 包括 `页面布局`, `路由注册`, `资源鉴权`, `列表组件`, `表单组件`,
`MockAPI` 等, 外部功能均以扩展的方式接入.
